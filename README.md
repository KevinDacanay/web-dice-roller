# Web Dice Roller

## Author
Kevin Dacanay

## Description
Web Dice Roller is a simple web application designed for Dungeons & Dragons gameplay. The site features a single button that rolls a 20-sided die (d20) and displays the result, simulating a physical die roll for quick and easy use during a game.

## Credits
All work for this project is original. It was built using HTML, CSS, and JavaScript to demonstrate basic random number generation and interactive web functionality.

## Instructions for Compiling and Executing

### Running Locally
This application consists of static HTML, CSS, and JavaScript files. No compilation is required to run the project locally.

1. Clone or download the repository to your local machine.
2. Navigate to the project directory.
3. Open the `index.html` file in any modern web browser (Chrome, Firefox, Edge, Safari).

### Deployment
This project is configured for continuous deployment using Azure Static Web Apps.

1. Changes pushed to the main branch of the GitHub repository automatically trigger a GitHub Actions workflow.
2. The workflow deploys the latest version of the site to the Azure Static Web App instance.

Visit the site @ [https://lemon-rock-0b7a8b61e.1.azurestaticapps.net/](https://lemon-rock-0b7a8b61e.1.azurestaticapps.net/)
